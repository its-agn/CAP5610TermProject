\section{Team Information}

\subsection{Team Members and Roles}
The project team consists of five members. Alexander Nardi served as the project coordinator. Each team member was responsible for one traditional machine-learning model and one deep-learning model, while the team jointly maintained the shared utilities, evaluation workflow, and final report.

\subsection{Responsibility Distribution}
Responsibilities were split by model family so that the full project covers five traditional models and five deep-learning models on the same Yelp review classification task. This distribution let each member develop one pair of models in depth while still contributing to a common experimental pipeline.
